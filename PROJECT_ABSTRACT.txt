FOODLE - Farm-to-Table Marketplace Platform


PROJECT OVERVIEW

Foodle is a comprehensive mobile marketplace application that connects local farmers directly with consumers, eliminating intermediaries and promoting sustainable, farm-fresh produce distribution. Built with React Native and Expo, the platform provides distinct, tailored experiences for two user types: farmers (sellers) and foodies (customers).


PROBLEM STATEMENT

Traditional agricultural supply chains involve multiple intermediaries, resulting in:
• Reduced profit margins for farmers
• Higher prices for consumers
• Delayed delivery and reduced freshness
• Limited transparency in product sourcing
• Difficulty for consumers to support local agriculture


SOLUTION

Foodle addresses these challenges by providing a direct-to-consumer platform with the following key features:

For Consumers (Foodies):
• Browse fresh, local produce from verified farmers
• Search and filter products by category, organic certification, and locality
• View detailed product information and farmer profiles
• Add items to cart with real-time inventory tracking
• Secure checkout with multiple payment options
• Order tracking from confirmation to delivery
• Save favorite products and farmers
• Rate and review purchases

For Farmers:
• Create and manage product listings with pricing and inventory
• Track sales analytics and earnings in real-time
• Manage customer orders with status updates
• Build customer relationships through profiles and ratings
• Access dashboard with key business metrics
• Receive notifications for new orders


TECHNICAL ARCHITECTURE

Technology Stack:
• Framework: React Native with Expo SDK
• Navigation: React Navigation (Native Stack Navigator)
• State Management: React Context API + Hooks
• Styling: Custom StyleSheet with responsive utilities
• Type Safety: TypeScript throughout the codebase
• UI Components: Native components with custom design system

Key Technical Features:
• User Context System for role-based UI rendering
• Responsive design with custom scaling utilities
• Safe area management for different device sizes
• Modal-based sidebar navigation
• Tab-based order management
• Dynamic routing based on user type
• Animated welcome screen with auto-navigation
